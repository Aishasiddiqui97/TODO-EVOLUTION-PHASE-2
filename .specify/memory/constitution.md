## Core Principles

### 1. User-Centric Design
The application must prioritize user experience, clarity, and accessibility.

### 2. Reliability & Security
All data handling must be secure, validated, and reliable.

### 3. Maintainable Code
Code should be clean, modular, and well-documented.

### 4. AI-Assisted Development
AI tools are used to accelerate development while keeping human oversight.

### 5. Iterative Delivery
Features are delivered in small, testable increments.
